# BLX Markets

Institutional marketing site — React, Vite, Tailwind CSS 4, GSAP (scroll transition + platform mock), Lenis.

## Run

```bash
npm install
npm run dev
```

## Design direction

- **IBM Plex Sans** — editorial, finance-appropriate typography
- Restrained palette: `#0f0f10` / `#f4f4f5` + accent `#cb2733` used sparingly
- No template cards, custom cursor, or infinite marquees
- Home: data table for accounts, grid markets, panel wipe (dark → light)

## Pages

`/`, `/account-types`, `/platform`, `/about`, `/faqs`, `/contact`
